# 자료형
메모리(RAM)에 데이터를 저장하기 위한 타입.
## Primitive Type의 구분
숫자(정수)-byte, short, int, long

숫자(부동수소점)-float, double

불린-bloolean

문자(한글자)-char
# 변수
데이터가 할당된 메모리 주소의 별칭

변수의 이름으로 메모리 주소 접근 가능

변수는 정의, 값 할당 부분으로 구성
## 변수의 정의
자료형 변수명(같은 이름으로 정의할 수 없음)

Ex) int number;
## 변수값 할당
이미 정의된 변수에 값을 할당할 때

Ex) number = 10;

변수의 정의와 할당을 동시에 할 수 있음

Ex) int number = 5;
